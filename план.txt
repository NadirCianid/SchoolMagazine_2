
Электронный школьный журнал старших классов.
Функции:
    Итоговые оценки (просмотр).
        1) Выбрать из бд список классов и поместить его в Список
        2) Показать всплывающее окно для выбора класса
        3) На основе выбранного класса запустить выборку всех учеников выбранного класса и сохранить их в Список
        4) Показать всплывающее окно для выбора ученика
        5) Поместить выбранные объекты в слоты (внизу справа)
        6) Заполнить таблицу итоговыми оценками выбранного ученика (завязать заполнение на метод, чтобы можно было его включить в кнопку ребута)

    Оценки (просмотр).
        1) Выбрать из бд список классов и список предметов и поместить их в Списки
        2) Показать всплыввающее окно для выбора класса и предмета
        3) Поместить выбранные объекты в слоты
        4) Заполнить таблицу оценками выбранного класса по выбранному предмету

    Домашнее задание (просмотр).
        1) Выбрать из бд список классов и список предметов и поместить их в Списки
        2) Показать всплыввающее окно для выбора класса и предмета
        3) Поместить выбранные объекты в слоты
        4) Выбрать из бд актуальные дз и заполнить ими таблицу

    Замечания (просмотр).
        1) Выбрать из бд список классов и список предметов и поместить их в Списки
        2) Показать всплыввающее окно для выбора класса и предмета
        3) Поместить выбранные объекты в слоты
        4) Выбрать из бд замечания и заполнить ими таблицу

    Дополнительные функции для учителя:
        Выставить оценку/ДЗ/Замечание:
            1) Открыть сцену для выставления оценки/ДЗ/замечания
            2) Считать введеные данные
            3) Выполнить запрос Insert
            4) Обновить текущую таблицу
